# 1 文档概述

## 1.1 文档目的

本文档为练遇健身小程序V2.0最终版产品需求文档，全面明确产品从用户端到教练端的全功能边界、业务逻辑、交互规范、视觉标准与验收准则，核心承接两大调整需求：一是彻底摒弃居家自主训练模式，转型**教练获客-私教预约-线下授课-课后复购**的全流程O2O闭环；二是打造与Keep完全区分的专属UI美术风格，避免视觉与定位同质化。文档作为设计、开发、测试、运营全团队的统一执行依据，保障产品按差异化定位精准落地，同时兼顾全文逻辑流畅、板块衔接自然，无上下文冲突。

## 1.4 核心术语说明

- **LBS精准匹配**：基于用户地理位置，优先推荐周边3-5km内符合需求的教练与课程，提升约课效率

- **双模式切换**：同一小程序，用户默认进入C端模式，完成教练资质认证后自动切换为B端教练模式，支持随时互切

# 2 产品整体规划

## 2.1 核心产品目标

1. **差异化定位目标**：彻底跳出Keep居家自主健身工具的定位，聚焦线下私教服务垂直赛道，打造微信生态内首款轻量化私教全闭环平台，解决用户“找教练难、选教练坑”与教练“获客难、管理难”的双向痛点；
2. **用户增长目标**：上线6个月内，入驻持证合规教练≥1000名，覆盖20个核心一二线城市，C端注册用户突破50万，日活用户突破5万；
3. **商业变现目标**：构建“私教服务佣金+付费会员+电商变现”三大盈利模型，上线6个月私教订单GMV突破1000万，平台佣金收入破100万，会员付费转化率≥8%；
4. **生态闭环目标**：打通“教练获客-用户约课-线下授课-课后服务-学员复购”全链路，叠加同城健身搭子社交、垂直电商配套，形成双向粘性生态。

## 2.2 产品整体架构（双端一体化）

产品采用微信小程序原生开发，支持双模式自动识别与自由切换，全局底层功能共用，端口功能差异化适配，核心架构如下：

- **全局底层功能**：微信授权登录、权限管理、消息推送、支付结算、内容审核、数据统计

- **C端用户模式（五大固定Tab）**：首页（教练推荐+核心入口）→今日（课程日程+课后管理）→运动（教练/课程检索）→商城（私教服务+实物商品）→我的（个人中心+教练入驻）

- **B端教练模式（五大专属Tab）**：首页（经营数据+待办课程）→学员（学员管理+服务跟进）→课程（课程上架+预约管理）→收益（订单结算+提现）→我的（资质管理+账号设置）

## 2.3 核心业务闭环流程

### 2.3.1 教练端全流程

教练提交入驻申请→资质实名认证与审核通过→个人主页装修→课程/课时/拼课活动上架→平台流量曝光+精准匹配学员→学员咨询/预约→教练确认订单→线下授课→扫码核销→课后点评与学员维护→学员复购/周期课续约

### 2.3.2 用户端全流程

微信一键登录→完善健身需求与个人信息→智能匹配同城教练→浏览教练主页与学员案例→1v1咨询沟通→预约体验课/周期私教课→线上支付费用→线下按时授课→扫码完成核销→提交课后评价→教练推送后续方案→课程复购/续课

# 3 UI美术风格差异化设计规范（核心区分Keep）

本章节为专属视觉规范，全程与Keep冷感极简工具风做彻底区分，主打**暖质专业社交风**，核心围绕“温暖、专业、连接、信任”打造视觉体系，所有页面设计必须严格遵循以下规范，杜绝同质化。

## 3.1 核心视觉对比（VS Keep）

|视觉维度|Keep原有风格|练遇健身差异化风格|
|---|---|---|
|核心定位|冷感居家工具，弱社交、强自主训练|暖感服务平台，强社交、强私教服务、强信任感|
|主色调|冷调绿色+黑白灰，低饱和冷感|活力暖橙（#FF7D3B）+深空灰（#2A2D34），高辨识度暖色调，视觉反差极强|
|视觉语言|扁平化、直角/小圆角、极简线条、无装饰|轻拟物、大圆角（16-24px）、微立体阴影、场景化实景，弱化工具感|
|核心素材|健身动作插画、矢量图形为主，极少真人图|真人教练授课实景、学员训练实拍为主，插画仅做辅助，强化真实感|
|排版逻辑|高密度信息堆砌、强网格、左对齐|充足留白、呼吸感排版、轻量化层级，突出教练与服务主体|
## 3.2 详细视觉规范

- **色彩规范**：主色#FF7D3B（核心按钮、品牌标识），辅助色#2A2D34（标题、正文），点缀色#36CFC9（数据、进度条），背景色#F8F9FA暖白，全程无冷色调大面积铺色；

- **组件规范**：所有卡片统一20px大圆角，轻微投影，无边框硬线；核心按钮16px大圆角，微渐变质感，拒绝扁平化直角按钮；

- **字体排版**：沿用微信原生字体，标题字重600-700，正文400，页面边距统一20px，模块间距24px，拒绝信息拥挤；

- **图标规范**：面性+线性结合，融入主色暖橙点缀，拒绝纯线性极简图标，提升视觉活力。

# 4 全局功能说明

## 4.1 登录注册与双模式切换

1. 登录方式：全程微信授权一键登录，无需手机号单独注册，降低用户与教练入驻门槛；
2. 双模式规则：默认进入C端用户模式，用户可通过“我的-教练入驻”入口提交资质，审核通过后自动解锁教练模式，支持随时手动切换端口，数据完全互通；
3. 信息完善：C端用户需完善健身目标、基础信息方可使用教练匹配功能；教练必须完成实名认证、资质认证、授课场地填写，审核通过后才能上架课程、接收订单。

## 4.2 权限管理规则

- 位置权限：用于同城教练、搭子、场馆匹配，未授权仅支持全城筛选，不影响核心功能使用；

- 相机/相册权限：用于头像上传、资质认证、扫码核销、评价晒图，按需授权；

- 通知权限：开课前24小时+1小时双推送提醒，订单、消息实时通知，引导授权不强制；

- 隐私权限：用户与教练位置信息全程脱敏，仅显示距离，不暴露精确地址，保障隐私安全。

## 4.3 全局导航与交互

1. C端底部固定Tab：首页、今日、运动、商城、我的（按用户要求保留命名与顺序），当前Tab高亮暖橙，点击切换，禁止左右滑动切换避免误触；
2. 教练端底部固定Tab：首页、学员、课程、收益、我的，贴合教练经营需求，与C端完全区分；
3. 全局交互：所有页面加载失败展示缺省页+重新加载按钮，核心操作二次确认，避免误操作。

## 4.4 消息与支付体系

1. 消息体系：分用户端（教练私信、课程提醒、订单通知）与教练端（学员咨询、核销提醒、收益到账），支持图文语音沟通，实时推送；
2. 支付体系：全程接入微信支付，符合官方安全规范，私教课程、实物商品分开结算，课程核销后结算教练收益，支持自动提现与手动提现。

# 5 C端五大核心页面详细需求

## 5.1 首页（Tab1，默认落地页）

**页面定位**：平台核心流量入口与教练曝光场，聚焦教练推荐、约课转化，彻底区别于Keep课程首页。
**布局从上至下**：顶部搜索+消息中心→轮播Banner（真人授课实景）→核心功能区（教练智能匹配、付费会员、同城搭子/拼课）→同城金牌教练横向推荐流→热门体验课/拼课专区→教练种草短视频信息流→底部预约体验课悬浮按钮。
**核心亮点**：首屏直接展示同城优质教练，突出“找私教”核心功能，无自主训练课程冗余推荐。

## 5.2 今日（Tab2）

**页面定位**：用户私教课程全周期管理页，围绕课前-课中-课后闭环设计，替代Keep自主训练计划页。
**布局从上至下**：日期切换栏→今日课程核心看板（待上课/已完成，开课前显示扫码核销入口）→课前准备提示→课后反馈与教练点评→身体数据追踪→快捷功能入口→每日签到。
**核心亮点**：上课倒计时、导航到店、联系教练一键直达，课后数据同步联动教练端，形成服务闭环。

## 5.3 运动（Tab3）

**页面定位**：教练与课程全量检索页，核心做私教服务匹配，弱化自主训练内容。
**布局从上至下**：搜索+多维度筛选（距离、价格、资质、评分）→核心场景入口（找教练、找团课、体验课专区、场馆预约）→教练分类导航→同城教练推荐列表→热门课程推荐。
**核心亮点**：按健身目标精准筛选教练，每款教练卡片展示资质、评分、体验课价格，降低用户决策成本。

## 5.4 商城（Tab4，对标Keep+适配私教）

**页面定位**：100%对标Keep商城交易、售后、运营体系，新增私教虚拟服务板块，实现“服务+实物”双电商模式。
**布局从上至下**：搜索+购物车+客服→轮播Banner→分类金刚位（前半部分私教包/团课卡/储值卡，后半部分完全对标Keep服饰、装备、补剂等实物）→核心服务营销区→实物商品专区→种草内容区→商品推荐流。
**核心规则**：实物商品交易、售后、物流完全照搬Keep标准；私教虚拟课程支持扫码核销、过期退款，核销后自动结算。

## 5.5 我的（Tab5）

**页面定位**：C端个人中心，整合课程、订单、会员、教练入驻入口。
**布局从上至下**：个人信息+会员标识→核心数据看板（上课次数、累计时长）→功能网格（我的课程、我的教练、拼课、优惠券等）→会员中心入口→教练入驻快捷入口→设置与客服。
**核心亮点**：一键跳转教练入驻通道，普通用户可快速转型教练，打通双向身份。

# 6 教练端核心功能与双模式切换规则

## 6.1 教练入驻与资质审核

1. 入驻门槛：年满18周岁，完成实名认证，持有国家健身教练资格证或国际权威认证，有固定授课场地；
2. 审核流程：提交材料→AI初审→人工1-2个工作日复审→审核通过→开通教练模式；
3. 教练等级：实习教练→认证教练→金牌教练→明星教练，等级越高流量扶持越多、佣金比例越低。

## 6.2 教练端五大Tab核心功能

- **首页**：经营数据看板（曝光、咨询、订单、收益）、今日待上课/待核销订单、学员咨询快捷入口、课程上架按钮；

- **学员**：学员分类管理（上课中/待跟进/已结课）、学员训练档案、身体数据、1v1沟通、复购提醒；

- **课程**：课程上架/编辑/下架、预约时间设置、拼课活动创建、核销管理；

- **收益**：收益统计、订单列表、提现管理、佣金规则、发票申请；

- **我的**：资质管理、教练主页装修、评价管理、平台规则、客服入口。

## 6.3 双模式切换细则

同一账号双身份互通，用户模式下的个人信息、会员权益在教练模式下同步保留；教练模式下的课程、收益数据独立展示，切换后不影响原有端口功能，无需重新登录，操作便捷。

# 7 私教全流程业务闭环规则

## 7.1 预约与确认规则

用户提交预约后，教练需12小时内确认，超时未确认订单自动取消，费用全额原路退还；用户可选择授课时间、场地，支持上门授课（需额外标注费用）。

## 7.2 取消与改期规则

开课前24小时以上免费取消改期；12-24小时取消扣除30%违约金；12小时内不支持取消；教练单方取消需全额退款并补偿用户优惠券，保障用户权益。

## 7.3 授课与核销规则

线下授课完成后，教练或用户均可扫码核销，核销后订单标记为已完成；未核销订单24小时内提醒，7天自动核销，核销后平台扣除约定佣金，收益T+7日可提现。

## 7.4 课后服务与评价规则

教练需24小时内提交课后点评、训练建议；用户可1-5星评分，评价公开展示，直接影响教练排名与流量；恶意评价平台审核后可删除，保障双方公平。

# 8 核心特色功能模块细则

## 8.1 教练智能匹配系统（原训练目标匹配升级）
